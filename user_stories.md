# User Stories – Patient Appointment Portal

## Admin User Stories
1. As an admin, I want to create and manage user accounts, so that I can control access to the system.
2. As an admin, I want to view all appointments, so that I can monitor system activity.
3. As an admin, I want to add or remove doctors, so that the system stays updated.

## Patient User Stories
1. As a patient, I want to register an account, so that I can access the system.
2. As a patient, I want to book an appointment, so that I can consult a doctor.
3. As a patient, I want to cancel or reschedule appointments, so that I can manage my schedule.

## Doctor User Stories
1. As a doctor, I want to view my appointments, so that I can prepare for my patients.
2. As a doctor, I want to update my availability, so that patients can book correctly.
3. As a doctor, I want to create prescriptions, so that patients receive proper treatment.
